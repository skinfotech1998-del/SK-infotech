Gallery Images Upload Instructions
====================================

1. Is "images" folder mein apni photos copy karo
2. Photos ko rename karo jaise:
   - project1.jpg
   - project2.jpg
   - project3.jpg
   etc.

3. Best image size: 800x600 pixels ya similar ratio
4. Supported formats: .jpg, .jpeg, .png, .webp

Example:
- images/project1.jpg
- images/project2.jpg
- images/cctv-installation.jpg
- images/biometric-system.jpg
